# S2 - Découverte de Wordpress


## Objectifs pédagogiques

* Configurer globalement un CMS
* Ajouter du contenu à un site internet
* Mettre en forme un contenu rédigé
* Organiser les contenus d’un site
* Créer des menus / liens
* Modifier l’apparence d’un site

## Critères d’évaluation

* Vérification des réglages de base
* Taxonomie (les tags et catégories demandées ont été créés)
* Structure d’url (les urls utilisent les noms de catégories / articles / pages)
* Indexation basique (Les métadonnées du site sont remplies)

**Les contenus dans Wordpress**

* Les pages demandées sont créées
* Les articles demandés sont créés
* Les contenus sont catégorisés / taggés comme demandé
* Les fichiers multimédias fournis (images, pdf etc.) sont intégrés

**Travailler avec les thèmes**

* Un autre thème que celui par défaut est utilisé
* Le thème est configuré pour correspondre à la charte graphique demandée
* Le thème utilise un menu comme demandé par le cahier des charges


## Modalité d’évaluation

* Créer un site vitrine pour un écrivain / critique à partir d’un cahier des charges
* Présenter son travail au client (l’animateur)

# S6/7 - Pratiquer les extensions

## Objectifs pédagogiques

* Découvrir les extensions Wordpress incontournables
* Comprendre le fonctionnement d’une boutique en ligne
* Comprendre les problématiques d’une boutique en ligne
* Identifier et comprendre les principes d’ergonomie
* Comprendre le référencement

## Critères d’évaluation

**Les extensions suivantes sont installées et configurées selon le cahier des charges**

* Akismet / Le site filtre les spams
* Page Builder / Les pages du site sont structurées selon le cahier des charges
* Woo Commerce / Les produits vendables sont créés et configurés, ainsi que les moyens de paiement demandés
* JetPack / Les produits sont partageables sur les réseaux sociaux
* SEO Pack / Les méta-données des produits sont bien renseignées


## Modalité d’évaluation

* Selon un cahier des charges
* Créer une landing page pour un produit
* Créer une boutique en ligne pour commander ce produit
* Configurer la boutique pour respecter les lois françaises (facture systématique, gestion de la TVA…)
* Optimiser sa boutique pour le référencement
* Validation par l’animateur sur le poste de l’apprenant
